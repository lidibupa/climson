climson (<b>C</b>ommand <b>L</b>ine <b>I</b>nterface <b>M</b>essaging <b>S</b>ervice lias<b>ON</b>)
=======

A simple, but powerful cli text messaging client that uses the C twilio api.

Installation
============
- run make 
- make install (as root)

Requirements
============
- twilio-c 
- libcurl-devel
